Side Channel Analysis
=====================

This chapter contains the analysis of selected algorithms in Botan with regard to side-channel leakage.
The methodology *Differential Address Trace Analysis* (DATA) and the framework of the same name are used for this purpose.
DATA was released at USENIX 2018 [DATA]_ and is being developed as an open-source project on GitHub [DATA_GIT]_.
The following chapters describe the methodology of DATA, the application in Botan and the results of the analysis.
